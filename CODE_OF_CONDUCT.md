# Code of Conduct

## Our Standards

We want this project to be a welcoming space for everyone. Be respectful, be constructive, be kind.

**Do:**
- Give and accept constructive feedback
- Focus on what's best for the project
- Be respectful of differing viewpoints and experiences

**Don't:**
- Use personal attacks, insults, or derogatory comments
- Troll, spam, or deliberately disrupt discussions
- Publish others' private information without permission

## Enforcement

Unacceptable behavior can be reported to **kevin@dishine.it**. All reports will be reviewed and addressed.
