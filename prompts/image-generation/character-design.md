# Character Design Prompt

Create detailed character concept art prompts for games, animation, illustration, and storytelling. This template structures character descriptions using concept art industry conventions—silhouette readability, color palettes, and design language—to produce visually distinctive, production-ready character concepts.

## When to Use

- Designing original characters for games, comics, or animated projects
- Generating character reference sheets for illustration pipelines
- Exploring visual concepts before commissioning a character artist
- Creating NPC or enemy designs for tabletop RPG campaigns
- Producing mascot or brand character concepts

**When NOT to use:**

- When you need multi-view turnaround sheets with precise proportions (requires specialized tools)
- For rigging-ready character models with T-pose requirements
- When the character must exactly match an existing copyrighted design

## The Technique

Strong character design prompts follow the concept art pipeline: start with the **character archetype** (role, personality), define the **visual design language** (sharp angles for villains, rounded shapes for friendly characters), specify the **art style** (cel-shaded, painterly, pixel art), set the **environment context**, describe **key distinguishing features** (silhouette-defining elements), and lock the **pose** (action, idle, portrait). The silhouette test is critical—a well-designed character should be recognizable from their outline alone.

### Design Language Principles

- **Rounded shapes**: Friendly, approachable, trustworthy characters
- **Angular/sharp shapes**: Aggressive, dangerous, villainous characters
- **Triangular silhouettes**: Powerful, heroic, imposing presence
- **Circular proportions**: Cute, young, non-threatening characters
- **Exaggerated proportions**: Stylized, cartoon-like, memorable at a glance
- **Realistic proportions**: Grounded, serious, narrative-driven characters

### Art Style Options

- **Painterly semi-realistic**: Visible brushwork, rich color depth, fantasy illustration feel
- **Cel-shaded/anime**: Bold outlines, flat color fills, stylized shading
- **Pixel art**: Retro game aesthetic, limited palette, blocky charm
- **Ink and wash**: High-contrast linework with watercolor-like shading

## Template

```
Character concept art of {{character_name}}, a {{character_type}}.
Art style: {{art_style}}.
Setting: {{setting}}.
Key features: {{key_features}}.
Pose: {{pose}}.
Full character visible, strong readable silhouette, distinctive color palette.
Detailed costume and accessory design, consistent proportions throughout.
Professional concept art quality, clean linework, polished rendering.
No background clutter, no extra limbs, no anatomical distortion.
```

## Tips

1. **Lead with silhouette** — Describe elements that create a unique outline: capes, horns, oversized weapons, or distinctive hats make characters instantly recognizable.
2. **Limit the color palette** — Specify 2–3 dominant colors plus one accent; this mirrors professional character design methodology and produces cohesive results.
3. **Describe personality through pose** — "Confident lean with crossed arms" tells the model about personality without requiring text annotations.
4. **Use art style references broadly** — Instead of naming specific artists, describe the technique: "painterly with visible brushstrokes" or "flat cel-shaded with bold outlines."
5. **Add asymmetrical details** — One glowing eye, a single mechanical arm, or a half-torn cloak adds visual interest and breaks generic symmetry.

## Common Mistakes

1. **Overloading with accessories** — Listing 10+ unique items creates visual noise; focus on 3–4 signature design elements that define the character.
2. **Vague archetype descriptions** — "A warrior" is too broad; "a battle-weary veteran knight with ceremonial scarring" gives the model clear visual direction.
3. **Ignoring proportions** — Not specifying body type or build leads to inconsistent anatomy; state "athletic build," "stocky frame," or "tall and lanky."
4. **Requesting multiple views in one prompt** — Turnaround sheets rarely work well; generate one strong angle first, then iterate with pose variations.
5. **Contradictory design language** — A "cute friendly villain" sends mixed signals; align shape language, color, and accessories with the intended character archetype.
