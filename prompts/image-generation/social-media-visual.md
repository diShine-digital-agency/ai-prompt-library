# Social Media Visual Creator

Generate scroll-stopping social media visuals optimized for specific platforms and content formats. This template combines platform-aware dimensions, trending visual styles, and brand consistency to produce marketing-ready graphics.

## When to Use

- Creating Instagram posts, stories, or carousel cover images
- Designing YouTube thumbnails or channel banners
- Producing Twitter/X header images or thread visuals
- Building Pinterest pins or Facebook ad creatives
- Generating consistent visual content for a content calendar

**When NOT to use:**

- When exact brand guideline compliance with vector assets is required
- For text-heavy graphics where typography precision matters
- When the final asset must be an editable layered design file

## The Technique

Social media visuals must balance **visual impact** at small sizes with **brand recognition** across posts. The key is specifying the platform constraints (aspect ratio, safe zones), a strong focal subject, a cohesive color story, and optional text overlay zones. Style modifiers like "flat illustration," "3D render," or "photography" set the visual language. Always design with mobile-first viewing in mind since most social consumption happens on phones.

### Platform Dimensions Quick Reference

- **Instagram Post**: 1080×1080 (1:1) or 1080×1350 (4:5)
- **Instagram Story/Reel**: 1080×1920 (9:16)
- **YouTube Thumbnail**: 1280×720 (16:9)
- **Twitter/X Post**: 1200×675 (16:9)
- **Pinterest Pin**: 1000×1500 (2:3)
- **LinkedIn Post**: 1200×627 (1.91:1)
- **Facebook Cover**: 820×312 (2.63:1)

### Visual Style Categories

- **Photography**: Real-world product shots, lifestyle scenes, portraits
- **Flat illustration**: Clean vector-style graphics, ideal for explainers
- **3D render**: Glossy, dimensional objects with dramatic lighting
- **Collage/mixed media**: Layered cutouts combining photo and graphic elements

## Tips

1. **Avoid AI-generated text** — Specify "no text in image" or "blank text zone" since AI models render typography poorly; add text in post-production.
2. **Design for the feed** — Images are seen at small sizes first; use bold shapes and high contrast that read clearly at 150×150 pixels.
3. **Include hex color codes** — Specifying exact hex values like "#FF6B6B" produces more consistent brand colors than vague descriptions.
4. **Leave breathing room** — Request "clean negative space" or "text-safe zone in upper third" to leave room for captions or CTAs.
5. **Batch with consistency** — When generating a series, repeat the same style and color palette keywords to maintain visual cohesion across posts.

## Common Mistakes

1. **Ignoring platform aspect ratios** — A square image cropped to 16:9 loses critical content; always specify the target dimensions.
2. **Too many focal points** — Social visuals need one hero element; crowded compositions lose impact at scroll speed.
3. **Relying on AI for text rendering** — Never expect clean, readable text from image generators; plan to composite text separately.
4. **Inconsistent brand colors** — Using vague color terms ("blue") across a series produces inconsistent results; always use hex codes.
5. **Forgetting safe zones** — Platform UI elements (like/comment buttons, profile overlays) obscure edges; keep critical content centered.
